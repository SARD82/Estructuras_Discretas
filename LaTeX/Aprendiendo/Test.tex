\section{Tablas Desde Overleaf}
Parte del desarrollo, aqui veras como insertar las tablas.

1. Primero vamos a Overleaf.
2. Creamos un archivo para hacer la tabla.
3. Desde el Visual Editor creamos la tabla en los tres puntitos de arriba a la derecha.
4. Ahi podemos ir creando la tabla moviendonos con "TAB" entre las celdas.
5. Desde el Code Editor copiamos el código y lo pegamos en VS Code.
6. Podemos usar los [H], etc...

8. Más sencillo podemos hacer tablas sencillas en Excel, copiarlas en Overleaf en el Visual Editor y despues copiar el código y pegarlo en VS Code.
9. Si es una tabla dificil, pegarla como imagen.
